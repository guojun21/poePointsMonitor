# 机器人统计组件优化

## 背景
用户希望 `poePointsMonitor` 中的“机器人统计”组件（`BotStatsCard`）能够学习 `csv-viewer` 中 `ModelStatCard` 的展示方式，支持排序和可视化条形图。

## 修改内容

### 1. `frontend/src/components/cards/StatsCard.jsx`
- **引入状态管理**：添加 `useState` 和 `useMemo`。
- **添加排序功能**：
  - 支持按“次数”、“总Cost”、“平均Cost”排序。
  - 添加 `SORT_OPTIONS` 常量。
  - 实现 `sortedBots` 计算逻辑。
- **添加可视化条形图**：
  - 计算最大值 `maxValue`。
  - 实现 `getBarWidth` 函数计算条形图宽度。
  - 渲染带有进度条的列表项。
- **优化展示信息**：
  - 显示排名、名称、次数（百分比）、总Cost、平均Cost。

### 2. `frontend/src/components/cards/StatsCard.css`
- **布局调整**：
  - `bot-stats-card` 改为 flex 布局，支持高度自适应。
  - 添加 `card-header-row` 实现标题和排序按钮的对齐。
- **样式新增**：
  - `.sort-toggle-group` & `.sort-toggle-btn`：排序切换按钮样式。
  - `.model-item` 系列：列表项、进度条背景、进度条颜色样式。
  - `.bot-rank-badge`：排名徽章样式。
  - `.custom-scrollbar`：自定义滚动条样式。
- **颜色适配**：
  - 使用 `linear-gradient` 为不同排名的机器人设置不同的进度条颜色。
  - 适配浅色主题（Light Mode），使用 `rgba(0, 0, 0, 0.05)` 作为背景色。

## 效果
- “机器人统计”卡片现在支持交互式排序。
- 数据展示更加直观，通过条形图可以快速对比不同机器人的使用情况。
- 保持了原有的卡片风格，同时融入了更丰富的数据可视化元素。
